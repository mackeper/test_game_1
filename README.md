# test_game_1

## Installation
Start with installing SFML:
http://www.sfml-dev.org/tutorials/2.4/start-linux.php

Then: 
```sh
$ git clone git@github.com:mackeper/test_game_1.git
$ cd test_game_1
$ make
```

## Usage
```sh
$ ./sfml-app
```
